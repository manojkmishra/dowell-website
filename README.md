## Project Information

This is Dowell's website developed on VueJS (v2.0+) and Vue CLI (v3). The aim of this project was to quickly develop a front-end only website to replace
Dowell's old, clunky and slow website which was developed on an out-dated Magento CMS.

[Live Preview](http://uat.website.dowell.com.au/)

## Technology

-   [VueJS 2.x](https://vuejs.org/v2/guide).
-   [Vue CLI 3.x](https://cli.vuejs.org/).

## Local Host Installation (Windows)

Prerequisites:

-   [Git Terminal](https://git-scm.com/downloads).
-   [NodeJS](https://nodejs.org/en/download/).

1.  Clone the repository:

        git clone https://github.com/<CHANGE THIS>

2.  Install front-end dependencies:

        npm install

3.  Compile front-end source (development):

        npm run serve

4.  Access application on web browser:

        http://localhost:8000

5.  Compile for PRODUCTION: (pre-rendering)

        npm run build

6. Point server to the /dist folder
